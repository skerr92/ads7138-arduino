Detailed Description

This is a library for the ADS7138 Analog To Digital Converter

Designed specifically to work with the Oak Development Technologies ADS7138 Acorn Analog to Digital Converter -—> urlhere

The sensor uses I2C to communicate with additional pins that can be connected to for other operational modes.
